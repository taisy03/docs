Get started with Noeome by uploading your first document and exploring your organization's structured knowledge.

## Prerequisites

Before you begin, you must have:

- A Noeome account — sign up at [noeome.com](https://www.noeome.com)
- Documents to upload (.mdx, .md, .txt, .pdf, .html, .docx)

## Get started

<Steps>
  <Step title="Create an account">
    Sign up at noeome.com using your work email.

    ```bash
    # Or via CLI if preferred
    noeome-cli signup
    ```
  </Step>
  <Step title="Upload documents">
    Drag and drop files or use our API to add documents.

    ```bash
    curl -X POST https://api.noeome.com/documents \
      -H "Authorization: Bearer $TOKEN" \
      -F "file=@meeting_notes.pdf"
    ```
  </Step>
  <Step title="Explore insights">
    Search across documents or view the knowledge graph.

    ```bash
    noeome-cli search "Q2 strategy"
    ```
  </Step>
</Steps>
