### Intelligent Document Processing
- Automatic indexing of uploaded documents

### Powerful Search

## Getting Started

<Steps>
  <Step title="Create an account">
    Sign up at [noeome.com](https://www.noeome.com) using your work email

    ```bash
    # Or via CLI if preferred
    noeome-cli signup
    ```
  </Step>
  <Step title="Upload documents">
    Drag and drop files or use our API to add documents

    ```bash
    curl -X POST https://api.noeome.com/documents \
      -H "Authorization: Bearer $TOKEN" \
      -F "file=@meeting_notes.pdf"
    ```
  </Step>
  <Step title="Explore insights">
    Search across documents or view the knowledge graph

    ```bash
    noeome-cli search "Q2 strategy"
    ```
  </Step>
</Steps>
