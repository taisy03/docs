Noeome connects to the tools your organization already uses to build its structured knowledge model.

## Available Integrations

<Card title="GitHub" icon="github">
  Connect your GitHub organization to index repositories, pull requests, commits, and issues. Automatically creates services and profiles your codebase.
</Card>
<Card title="Slack" icon="slack">
  Connect Slack channels to include conversations and decisions in your knowledge graph. Publish broadcasts directly as formatted Slack messages.
</Card>
<Card title="LinkedIn" icon="linkedin">
  Publish generated content as LinkedIn posts. Uses OAuth to post as your organization's LinkedIn page.
</Card>
<Card title="Mintlify" icon="book-open-cover">
  Publish broadcast content as documentation PRs to your Mintlify docs repository. Configure which docs repo and section to target.
</Card>
<Card title="Linear" icon="square-terminal">
  Connect Linear to index projects, issues, and team workflows into the knowledge graph.
</Card>
<Card title="Mailchimp" icon="envelope">
  Publish broadcast content as email draft campaigns via the Mailchimp API.
</Card>

## Integration Management

Integrations are configured in the **Admin** settings page. Each integration requires:

1. **Authorization** — OAuth flow or API key for the provider
2. **Verification** — Noeome validates the connection is working
3. **Configuration** — Provider-specific settings (e.g., docs repo for Mintlify)
